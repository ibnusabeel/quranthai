# คู่มือการใช้งาน Keystatic CMS - สารานุกรมตัฟซีร

## 🎯 ภาพรวม

ตอนนี้เว็บไซต์ได้รับการอัพเดทให้อ่านข้อมูลจากไฟล์ที่สร้างใน Keystatic โดยอัตโนมัติแล้ว!

## 📝 วิธีสร้างเนื้อหา

### 1. เข้าสู่ Keystatic CMS

เปิดเบราว์เซอร์และเข้าไปที่:
```
http://127.0.0.1:4322/keystatic
```

### 2. สร้างหนังสือตัฟซีรใหม่

1. คลิกที่ **"Tafsir Books"** ในแถบด้านซ้าย
2. คลิก **"Create Entry"**
3. กรอกข้อมูล:
   - **Title**: ชื่อภาษาอังกฤษ (เช่น `tafsir-as-sadi`)
   - **Title (Thai)**: ชื่อภาษาไทย (เช่น `ตัฟซีร อัส-สะอดี`)
   - **Title (Arabic)**: ชื่อภาษาอาหรับ (เช่น `تفسير السعدي`)
   - **Author**: ชื่อผู้แต่ง (เช่น `อับดุรเราะห์มาน อัส-สะอดี`)
   - **Author (Arabic)**: ชื่อผู้แต่งภาษาอาหรับ
   - **Description**: คำอธิบายเกี่ยวกับหนังสือ (สามารถใช้ rich text ได้)
   - **Cover Image URL**: URL รูปปก (ถ้ามี)
4. คลิก **"Create"**

### 3. สร้างเนื้อหาตัฟซีร (Tafsir Entry)

1. คลิกที่ **"Tafsir Entries"** ในแถบด้านซ้าย
2. คลิก **"Create Entry"**
3. กรอกข้อมูล:
   - **Slug**: ชื่อไฟล์ (เช่น `alfalaq`, `alfatihah`)
   - **Book Slug**: ชื่อหนังสือ (ต้องตรงกับ Title ของหนังสือที่สร้างไว้ เช่น `tafsir-as-sadi`)
   - **Surah Number**: เลขที่ซูเราะห์ (เช่น `1` สำหรับอัล-ฟาติหะฮ์, `113` สำหรับอัล-ฟะละก)
   - **Surah Name (Thai)**: ชื่อซูเราะห์ภาษาไทย (เช่น `อัล-ฟาติหะฮ์`)
   - **Surah Name (Arabic)**: ชื่อซูเราะห์ภาษาอาหรับ (เช่น `الفاتحة`)
   - **Ayah Start**: อายะฮ์เริ่มต้น (เช่น `1`)
   - **Ayah End**: อายะฮ์สิ้นสุด (เช่น `7`)
   - **Tafsir Content**: เนื้อหาตัฟซีร (ใช้ MDX editor)
4. คลิก **"Create"**

## 📂 โครงสร้างไฟล์

เมื่อสร้างเนื้อหาผ่าน Keystatic ไฟล์จะถูกบันทึกที่:

- **Books**: `src/content/books/[title].mdx`
- **Tafsir**: `src/content/tafsir/[slug].mdx`

## 🔄 การแสดงผลบนเว็บไซต์

หลังจากสร้างเนื้อหาแล้ว:

1. **หน้าแรก** (`/`): จะแสดงรายการหนังสือตัฟซีรทั้งหมด
2. **หน้าหนังสือ** (`/tafsir/tafsir-as-sadi`): จะแสดงรายการซูเราะห์ทั้งหมดของหนังสือนั้น
3. **หน้าอ่านตัฟซีร** (`/tafsir/tafsir-as-sadi/alfalaq`): จะแสดงเนื้อหาตัฟซีรของซูเราะห์นั้นๆ

## ⚠️ ข้อควรระวัง

### 1. ชื่อ Slug ต้องตรงกัน
**Book Slug** ในTafsir Entry ต้องตรงกับ **Title** ของ Book ที่สร้างไว้

ตัวอย่าง:
- Book Title: `tafsir-as-sadi`
- Tafsir Entry Book Slug: `tafsir-as-sadi` ✅

### 2. ไฟล์ Extension
Keystatic จะสร้างไฟล์เป็น `.mdx` (ไม่ใช่ `.mdoc`)

### 3. การรีเฟรชเว็บไซต์
หลังสร้างเนื้อหาใหม่ อาจต้องรีเฟรชเพจเว็บไซต์เพื่อให้เห็นการเปลี่ยนแปลง

## 🎨 การจัดรูปแบบเนื้อหา (MDX)

Keystatic รองรับ Markdown และ MDX:

```markdown
## หัวข้อใหญ่
### หัวข้อรอง

**ตัวหนา**
*ตัวเอียง*

- รายการแบบ bullet
- รายการที่ 2

1. รายการแบบตัวเลข
2. รายการที่ 2

[ลิงก์](https://example.com)

> คำพูดที่ยกมา
```

## 💡 เคล็ดลับ

1. **ใช้ Surah Number**: เพื่อให้ซูเราะห์เรียงลำดับถูกต้อง
2. **เขียนเนื้อหาแบบ Markdown**: ใช้ heading, list, bold เพื่อให้อ่านง่าย
3. **ตั้งชื่อ Slug ที่ชัดเจน**: เช่น `alfatihah`, `albaqarah` เพื่อให้ URL อ่านง่าย

## 🚀 ขั้นตอนถัดไป

1. สร้างหนังสือตัฟซีรแรก
2. เพิ่มซูเราะห์ทีละซูเราะห์
3. เขียนเนื้อหาตัฟซีรให้สมบูรณ์
4. ตรวจสอบผลลัพธ์บนเว็บไซต์
